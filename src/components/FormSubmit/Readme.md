A button for a [Form](#form).

For best results, insert it inside a [FormButtons](#formbuttons) container.

```jsx
import Bluelib from "../Bluelib";
import Form from "../Form";


<Bluelib>
    <Form>
        <Form.Buttons>
            <FormSubmit>Submit</FormSubmit>
        </Form.Buttons>
    </Form>
</Bluelib>
```
