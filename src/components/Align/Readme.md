A `<div>` which changes the alignment of the contained text using the `align-*` Bluelib class.

```jsx
import Bluelib from "../Bluelib";

<Bluelib>
    <Align value={"left"}>
        Left
    </Align>
    <Align value={"center"}>
        Center
    </Align>
    <Align value={"right"}>
        Right
    </Align>
</Bluelib>
```
